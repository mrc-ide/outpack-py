outpack.copy_files.location_build_pull_plan(packet_ids: List[str], locations: List[str] | None, *, files: Dict[str, List[str]] | None = None, recursive: bool, root: outpack.root.OutpackRoot) → LocationPullPlan

Create a plan to pull packets from one or more locations.

Parameters

packet_ids :

A list of packet IDs to pull.

locations :

A list of location names from which to pull packets. If None, all configured locations will be considered.

files :

A filter restricting, for each packet, which file hashes to pull. This allows a subset of a packet’s files to pulled. If None, or if a packet ID is mising from the dictionary, the entire packet is pulled.

recursive :

If True, all transitive dependencies of the requested packets will be pulled as well.

root :

The root object used to determine the location configuration and which files are missing and need pulling.
